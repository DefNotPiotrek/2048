### Prerequisites
* **Java JDK** (Version 8 or newer) must be installed.

### File Structure
Ensure the font file is placed in the correct resource directory:
/src/pl/chwilaprogramowaniadladebila/  <-- .java source files
/res/                                    <-- CourierPrimeSans.ttf

### Compilation and Running

Navigate to the project's root directory (parent of `src`) in your terminal:

1. **Compile:**
   javac -d bin src/pl/chwilaprogramowaniadladebila/*.java

2. **Run:**
   java -cp bin pl.chwilaprogramowaniadladebila.Main

*(Alternatively, open the project in an IDE like IntelliJ IDEA or Eclipse and run the Main class).*
